# 逻辑简述

本文概述当前“图框识别与图签提取”的业务管线，作为参数规范与实现的一致性备忘。

## 1) 总体思路（几何优先 + 锚点ROI验证）

- **几何候选优先**：在指定图层范围内先建立外框候选。
- **纸张拟合确定比例**：以几何拟合得到的 `sx/sy` 作为图纸比例依据。
- **锚点用于验证与选择**：锚点文本仅用于 ROI 验证和候选优选，不作为 RB 定位入口。
- **A4(SMALL5)簇扩展**：锚点命中的 A4 作为种子扩展同簇外框。

## 2) 处理流水线（与参数规范一致）

1. **DWG→DXF**：输入转换。
2. **几何候选（layer_priority 内）**：
   - 闭合且轴对齐的 `LWPOLYLINE/POLYLINE` 作为优先矩形候选。
   - 开放但 **4 点且轴对齐** 的多段线可作为矩形候选。
   - 其它开放多段线拆线段参与 LINE 重建。
   - LINE 重建仅在**未找到 polyline 候选**时执行，并按**每个 layer 独立**进行；`line_rebuild_limits`（含 `max_coord_pairs`）对单层生效，避免跨层坐标组合爆炸导致整体重建失效。
3. **纸张拟合**：对候选矩形执行 `PaperFitter.fit_all`，得到 `paper_variant_id/sx/sy/roi_profile_id`。
4. **锚点 ROI 验证**：
   - 用 `anchor_roi_rb_offset_1to1`（含项目覆盖）还原 ROI。
   - ROI 内命中锚点文本则候选有效，按 `fit_error` 取最优。
5. **SMALL5(A4)簇扩展**：
   - 以锚点命中的 A4 为种子扩展同簇外框。
6. **字段提取与比例一致性校验**：属于模块3（图签字段提取），在模块2中不执行。

## 3) 比例确定方式

- **图纸比例以几何拟合为准**：`sx/sy`（或 `geom_scale_factor`）来自纸张拟合。
- **锚点字高推导比例仅用于 RB 兼容模式**（`frame_detect_mode=rb_anchor`）；
  在当前“几何优先”模式中不参与比例确定。
