## 1) 已实现功能

### 1.1 核心类

- **`A4MultipageGrouper`**（`backend/src/cad/a4_multipage.py`，291行）
  - 实现接口：`IA4MultipageGrouper`
  - 唯一公开方法：`group_a4_pages(frames: list[FrameMeta]) -> tuple[list[FrameMeta], list[SheetSet]]`
  - 构造参数：`spec_path: str | None = None`（默认从 `documents/参数规范.yaml` 加载）

### 1.2 功能清单

| 功能 | 方法 | 说明 |
|---|---|---|
| A4 判定 | `_is_a4_frame()` | 读取 `paper_variant_id`（含 `"A4"` 即为 A4），不重新检测 |
| 簇构建 | `_build_clusters()` | 按间距连边 + DFS 取连通分量；`gap_threshold_factor` 从 YAML 读取（默认 1.0） |
| 邻接判断 | `_frames_are_neighbors()` | `dx < threshold AND dy < threshold`，threshold = factor × min(W,H) |
| Master 定位 | `_identify_master()` + `_calculate_master_score()` | 按字段命中评分（engineering_no/internal_code/external_code/page_total 各+1，page_index==1 额外+2） |
| 簇处理 | `_process_cluster()` | 整合 Master 识别、字段检查、张数校验、页面构建、一致性校验 |
| Master 缺失字段检查 | `_check_master_fields()` | `internal_code/external_code/title_cn/page_total` 缺任一 → flag `A4多页_Master缺关键字段` |
| 辅助张数校验 | `_check_page_count_consistency()` | Master/Slave 声称的 page_total 与实际帧数不一致 → flag `A4张数有误，请检查` |
| 一致性校验 | `SheetSet.validate_consistency()` | 页码重复/不连续/首页异常 → 对应 flags |
| 元数据继承 | `SheetSet.get_inherited_titleblock()` | 返回 Master 的 titleblock 字段 dict |

### 1.3 处理流程

```
group_a4_pages(frames)
  │
  ├─ 1. 分离 A4 / 非 A4 图框
  │     └─ A4 < 2 个 → 直接返回原始 frames，不成组
  │
  ├─ 2. _build_clusters(a4_frames)
  │     └─ 对所有 A4 帧按间距连边，DFS 取连通分量
  │
  ├─ 3. 对每个簇（≥2 帧）调用 _process_cluster(cluster)
  │     ├─ _identify_master() → 评分选 Master
  │     ├─ _check_master_fields() → 关键字段缺失 flag
  │     ├─ page_total = len(cluster)（实际帧数 = 权威值）
  │     ├─ _check_page_count_consistency() → 辅助张数校验
  │     ├─ 构建 PageInfo 列表（Master + Slave 均保存 frame_meta）
  │     ├─ 按 page_index 排序
  │     └─ validate_consistency() → 页码连续性等校验
  │
  └─ 4. 返回 (remaining_frames, sheet_sets)
        ├─ remaining: 非 A4 帧 + 未成组的 A4 帧 → 传给模块5 逐帧裁切
        └─ sheet_sets: A4 多页成组 → 传给模块5 多页裁切
```

### 1.4 page_total 取值规则（重要）

**真实多页张数 = 簇内检测到的实际 A4 图框数量 `len(cluster)`**

Master 图签区的"共N张 第M张"和 Slave 右上角页码标记的 page_total 都**仅为辅助校验信息**，不作为 page_total 的来源。三者一致时无 flag；任一不一致则追加 `A4张数有误，请检查`（不中断，预留给前端展示）。

### 1.5 Flag 清单

| Flag | 产生位置 | 触发条件 |
|---|---|---|
| `A4多页_Master缺关键字段` | `_check_master_fields()` | Master 缺 internal_code/external_code/title_cn/page_total 之一 |
| `A4张数有误，请检查` | `_check_page_count_consistency()` | Master 或 Slave 的 page_total 与实际帧数不一致 |
| `A4多页_页数不一致` | `validate_consistency()` | 安全检查：len(pages) != page_total（正常不触发） |
| `A4多页_页码重复` | `validate_consistency()` | 两个 PageInfo 的 page_index 相同 |
| `A4多页_页码不连续` | `validate_consistency()` | 页码不是 [1,2,...,N] 的完整序列 |
| `A4多页_首页页码异常` | `validate_consistency()` | Master 的 page_index != 1 |

**所有 flag 只追加，不中断处理流程。** `A4张数有误，请检查` 需预留前端接口传给用户。


### 3.2 模块5 使用 SheetSet 的关键字段

| 字段路径 | 用途 |
|---|---|
| `sheet_set.pages[]` | 按 page_index 升序的页面列表 |
| `sheet_set.pages[i].outer_bbox` | 每页的裁切窗口坐标（`BBox: xmin/ymin/xmax/ymax`） |
| `sheet_set.pages[i].frame_meta` | **Master 和 Slave 均有引用**，可访问完整 FrameMeta |
| `sheet_set.pages[i].has_titleblock` | `True` = Master 页，`False` = Slave 页 |
| `sheet_set.page_total` | 总页数（= 实际帧数） |
| `sheet_set.master_page` | 指向 Master 的 PageInfo（= pages 中 has_titleblock=True 的那个） |
| `sheet_set.master_page.frame_meta.titleblock` | 整组的元数据来源（用 `get_inherited_titleblock()` 获取） |
| `sheet_set.flags` | 已收集的告警（模块5 应透传到 manifest） |

### 3.3 模块5 裁切 A4 多页的建议流程

```
split_sheet_set(dxf_path, sheet_set, output_dir):
  for page in sheet_set.pages:
      clip_bbox = page.outer_bbox          # 裁切窗口
      # 按 clip_bbox 做窗口裁切（保留与 bbox 相交的实体）
      # 坐标不平移、不归零（参见 a4_multipage.clipping 配置）

  # 输出方式（参见 a4_multipage.output 配置）:
  #   优先: 多页 PDF（每页一个 clip_bbox 打印窗口，按 page_index 排序合并）
  #   兜底: union bbox 单页大图 PDF
  #   DWG: 保持各 A4 外框相对位置不变
```

### 3.4 YAML 配置位置（模块5 使用）

| 参数路径 | 用途 |
|---|---|
| `a4_multipage.clipping.margin_percent` | 裁切边距（1%~2%） |
| `a4_multipage.clipping.entity_rule` | 实体保留规则（bbox 相交则保留） |
| `a4_multipage.clipping.coordinate_rule` | 坐标规则（不平移、不归零） |
| `a4_multipage.output.pdf.preferred` | 优先多页 PDF |
| `a4_multipage.output.pdf.method` | 逐页窗口打印后合并 |
| `a4_multipage.output.fallback` | 兜底为 union bbox 单页大图 |
| `a4_multipage.output.dwg` | DWG 保持相对位置 |

## 4) 与任务守则的偏差说明

以下功能点的实现与 `模块4_任务守则.md` 的原始描述有所调整，均基于真实 DXF（1818仿真图.dxf）验证后的用户反馈：

| 守则原文 | 实际实现 | 原因 |
|---|---|---|
| `page_total` 来自 Master.titleblock.page_total | `page_total = len(cluster)`（实际帧数） | Master 图签区"共N张"记录的是文件级信息（如 =1），非多页组张数；Slave 页码标记也只是辅助信息 |
| flag `A4多页_页总数冲突`（Slave vs Master） | flag `A4张数有误，请检查`（Master 或 Slave vs 实际帧数） | 权威值改为实际帧数后，校验对象随之调整 |
| `page_total < 2` 回退 | 簇 ≥ 2 帧时始终成组（page_total = cluster size ≥ 2） | page_total 取实际帧数后，簇内 ≥ 2 帧自然满足条件 |
