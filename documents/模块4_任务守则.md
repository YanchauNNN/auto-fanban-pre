## 0) 任务边界与前置条件

### 输入已具备（模块2+3的产物）
- `list[FrameMeta]`：所有图框，每个已包含：
  - `.runtime.paper_variant_id`：纸张类型（含 `"A4"` 的为 A4 图框）
  - `.runtime.outer_bbox`：外框坐标
  - `.runtime.sx / .sy`：缩放因子
  - `.runtime.flags[]`：已有告警
  - `.titleblock.page_total / .page_index`：**A4主帧已由模块3填充**
  - `.titleblock.internal_code` 等：**A4主帧有值，从属帧为 None**

### 模块3已完成的A4相关工作（重要！不要重复）
- A4主帧（Master）：完整 titleblock 提取
- A4从属帧（Slave）：**右上角页码已提取**（`page_total/page_index`），存放在 `frame.titleblock` 和 `frame.raw_extracts["A4_page_marker"]` 中
- 判定依据：有锚点文本 → 主帧；无锚点文本 → 从属帧（仅提取页码）

### 本模块输出
- `tuple[list[FrameMeta], list[SheetSet]]`
  - 第一项：非A4多页的图框（含单独A4/非A4图框），直接传给模块5
  - 第二项：A4多页成组列表（`SheetSet`），由模块5统一裁切输出

## 3) 必须满足的功能点（Acceptance）

### 3.1 触发条件（何时启用A4多页规则）
```
同一DXF中:
  ≥2 个 A4 外框 属于同一空间簇
  且 簇内至少 1 个 Master Page（锚点命中 + 关键字段命中数≥3）
```

### 3.2 A4 判定
- 已由模块2完成（`paper_variant_id` 含 `"A4"`）
- **模块4直接读取，不需要重新判定**

### 3.3 簇构建
```
gap_threshold = gap_threshold_factor × min(W_obs, H_obs)
连边条件: min_dx < gap_threshold AND min_dy < gap_threshold
取连通分量
```
- `gap_threshold_factor` 从 YAML `a4_multipage.cluster_building.gap_threshold_factor` 读取（默认 1.0）

### 3.4 Master 定位
```
评分规则:
  engineering_no 非空 → +1
  internal_code 非空 → +1
  external_code 非空 → +1
  page_total 非空 → +1
  page_index == 1 → +2

选择: 最高分
并列时: 选能解析出 page_total 且 page_index=1 的
```

### 3.5 Slave 页码处理
- **模块3已提取 Slave 的 `page_total/page_index`**（来自 `_extract_a4_page_marker` + `_fallback_a4_page_marker`）
- 模块4只需**读取并校验**，不需要重新提取
- 若 Slave 的 `page_index` 为 None → flag `A4多页_页码缺失`

### 3.6 一致性校验（对照 YAML `consistency_checks`）

| 校验项 | Flag 名称 | 说明 |
|---|---|---|
| 簇内A4外框数 ≠ Master.page_total | `A4多页_页数不一致` | 实际帧数与声明不符 |
| 页码重复 | `A4多页_页码重复` | 两帧 page_index 相同 |
| 页码不连续 | `A4多页_页码不连续` | 如 1,2,4 缺 3 |
| Slave 的 page_total ≠ Master 的 page_total | `A4多页_页总数冲突` | Slave 右上角声明与 Master 不一致 |
| Master 的 page_index ≠ 1 | `A4多页_首页页码异常` | Master 页码不是第1张 |
| Master 缺 internal_code/external_code/title_cn/page_total | `A4多页_Master缺关键字段` | 参见 `metadata.missing_master_fields_policy` |

**所有校验：只追加 flags，不中断处理！**

### 3.7 元数据继承
- 整组 SheetSet 的元数据取自 Master 的 titleblock
- 不从 Slave 回填（已确认取消回填策略）
- 通过 `SheetSet.get_inherited_titleblock()` 供后续模块消费

### 3.8 page_total < 2 回退
- 若 Master 的 `page_total < 2`（或为 None），该簇不按多页处理
- 将这些帧退回到"普通A4单页"列表

## 4) 输出结构（`SheetSet`）

```python
SheetSet(
    sheet_set_type="A4_MULTI_PAGE_001",
    paper="A4",
    cluster_id="<uuid>",
    page_total=7,                    # 来自 Master.titleblock.page_total
    pages=[
        PageInfo(page_index=1, outer_bbox=..., has_titleblock=True,  frame_meta=master_frame),
        PageInfo(page_index=2, outer_bbox=..., has_titleblock=False, frame_meta=slave_frame_2),
        PageInfo(page_index=3, outer_bbox=..., has_titleblock=False, frame_meta=slave_frame_3),
        ...
    ],
    master_page=<pages[0]>,          # 指向 page_index=1 的 PageInfo
    flags=["A4多页_页码不连续", ...]  # 校验产生的告警
)
```

### 关键约定
- `pages` 按 `page_index` 升序排列
- `master_page` 指向 has_titleblock=True 的那个 PageInfo
- **Slave 的 `frame_meta` 也应该保存引用**（模块5裁切时需要 outer_bbox）
- flags 只追加，不清空
